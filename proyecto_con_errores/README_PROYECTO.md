# Proyecto con errores: Lúmina Rooftop Bar

Esta carpeta contiene el proyecto que debes corregir.

## Cómo abrirlo

1. Abre esta carpeta en Visual Studio Code.
2. Abre `index.html`.
3. Ejecuta el proyecto con Live Server o abre `index.html` en Google Chrome.
4. Revisa la consola del navegador.
5. Corrige errores en HTML, CSS y JavaScript.

## Archivos principales

- `index.html`: estructura del sitio.
- `css/styles.css`: estilos visuales.
- `js/app.js`: interacciones, filtros y formulario.
- `assets/`: imágenes SVG del proyecto.

## Importante

El proyecto tiene errores intencionales. No borres todo el código. Diagnostica y corrige de forma ordenada.
