# Actividad 10 - Caza errores: Lúmina Rooftop Bar

## Propósito

Este paquete contiene una actividad práctica para la sección **Errores frecuentes en HTML y CSS; diagnóstico, corrección y documentación** de la Unidad 2. La actividad está diseñada para que el estudiante abra un proyecto web en Visual Studio Code, lo ejecute en Google Chrome, encuentre errores intencionales, los corrija y documente el proceso.

## Uso sugerido para el docente

1. Entregar a los estudiantes únicamente estos archivos y carpetas:
   - `README_ESTUDIANTE.md`
   - `CUESTIONARIO_ESTUDIANTE.md`
   - `GUIA_MAPA_MENTAL_NOTAS.md`
   - `PLANTILLA_BITACORA_CORRECCION.md`
   - `PLANTILLA_EVIDENCIAS.md`
   - `RUBRICA_EVALUACION.md`
   - `INSTRUCCIONES_GIT_ESTUDIANTES.md`
   - `proyecto_con_errores/`
   - `evidencias/`

2. No entregar al estudiante la carpeta:
   - `solucion_docente/`

3. Configurar Moodle como **Tarea (Assignment)**.

4. Solicitar como entrega:
   - ZIP del proyecto corregido.
   - Bitácora de corrección.
   - Cuestionario resuelto.
   - Mapa mental, mapa conceptual o notas técnicas.
   - Evidencias en capturas: antes, después, consola y código.

## Resultado de aprendizaje esperado

El estudiante ejecuta y adapta código, reproduce y modifica fragmentos, realiza pruebas, corrige errores frecuentes en HTML, CSS y JavaScript, y documenta técnicamente las soluciones aplicadas.
